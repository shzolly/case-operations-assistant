# Family Case Follow-Up Procedure

Source ID: SOP-FM-001
Domain: Family

When a family case has no docket activity for more than 14 days, staff should
verify the latest filing status, review missing documents, and create an
internal follow-up task for the assigned unit.

Staff should check whether certifications, notices, or required supporting
documents are outstanding before taking action. If the matter involves a minor,
staff should follow the additional unit review process before external
communication is sent.

External communication requires supervisor approval.
